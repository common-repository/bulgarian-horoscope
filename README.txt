== Description ==

С този плъгин можете лесно да сложите хороскоп във вашият Wordpress сайт.
Без усилия, просто инсталирайте и сложете джаджата където искате да се покаже хороскопа.
Хороскопите се пишат на сайта https://znacite.com, плъгина ги чете от там и ги показва на избрано от вас място във вашият Wordpress сайт.

При всякакви проблеми или въпроси можете да пишете на horoscop@znacite.com

== Installation ==

1. Качете плъгина в /wp-content/plugins
2. Активирайте плъгина чрез секция "Плъгини"
3. Добавете джаджата в някое от полетата за джаджи в сайта ви

== Frequently Asked Questions ==
Кога се обновяват хороскопите - В началото на всеки ден 00:00 плъгина взима новият хороскоп за деня.
